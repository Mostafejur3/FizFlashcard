Configure Supabase Connection
Setup Instructions
  1. Create a new project at supabase.com
  2. Go to Settings > API in your Supabase dashboard
  3. Copy your Project URL and anon/public key
  4. Paste them in the form above and click "Save Configuration"
  5. Test the connection to ensure everything is working

Note: The service key is optional and only needed for admin operations.

Database Connection Details (stored securely on your backend)
How this works
  1. Deploy your small backend API connected to your hosting database.
  2. Enter the API Base URL and API Key here and click "Save".
  3. Optionally fill DB host, port, name, user, password, and type for the backend to test.
  4. Click "Test Connection" to verify the API is reachable and can connect to your DB.

This does not change current Supabase behavior. It only stores settings for a future switch.
